✨ Features

🏖️ Tour package sections (Hawaii, Goa, Dubai, Singapore)

🎨 Clean and simple UI

📸 Attractive images of destinations

📱 Beginner-friendly and easy to customize

🛠️ Technologies Used

HTML5
